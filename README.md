# WeatherNow - Premium Flutter Weather Application

WeatherNow is a premium, real-time weather forecasting application built using Dart and Flutter. It is designed following Clean Architecture principles, leveraging modern styling, rich Glassmorphism visual components, and robust, hardware-linked system permissions and location checking pipelines.

## 🚀 Quick Start Guide

### Requirements
- **Flutter SDK**: `^3.11.3`
- **Dart SDK**: `^3.11.3`

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/weather_app.git
   cd weather_app
   ```
2. Fetch dependencies:
   ```bash
   flutter pub get
   ```
3. Run the application:
   ```bash
   flutter run
   ```

## 🎨 UI/UX Features
- **Modern Glassmorphism Design**: High-fidelity glass container UI components offering premium visual appeal.
- **Hardware Integration**: Live location tracking using Geolocator and runtime permission management with Permission Handler.
- **WeatherNow Splash Screen**: Custom gradient launch screen featuring smooth scale and fade logo animations.
- **Dynamic Weather Mood Gradients**: Background styling updates automatically depending on weather conditions (sunny, rainy, snowy, misty, stormy).
- **Responsive Layout**: Designed to dynamically adapt layouts for mobile phones and tablet devices.
